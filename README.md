# song-recommendation
A java microservice that provides Spotify song recommendation

## Tech Stack

- Java 21   
- Maven
- Docker
- MySQL
- kubernetes

## CICD

- GitHub Actions to build and test the application
- GitHub Actions to deploy the application to aws eks


## Database

- aws rds mysql
